# Nith Digital

Static website for **Nith Digital** — websites, data & apps for businesses in Dumfries & Galloway.

## About

Nith Digital builds modern websites, interactive dashboards, and custom tools for small businesses and organisations across D&G. Based in Sanquhar, serving all of Dumfries & Galloway and beyond.

**Live site:** [nithdigital.uk](https://nithdigital.uk)

## Structure

```
index.html          # Homepage (single-page app with hash routing)
404.html            # Custom 404 page with auto-redirect
robots.txt          # Search engine directives
sitemap.xml         # Sitemap for SEO
favicon.ico         # Browser favicon
apple-touch-icon.png # Apple touch icon
```

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- Google Fonts (Libre Baskerville + DM Sans)
- No build step required — deploy as-is

## Deployment

This is a static site. Deploy to any static hosting provider:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel
