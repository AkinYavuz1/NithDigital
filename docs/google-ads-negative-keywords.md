## Notes on Match Types Used

| Match Type | When Used in This List | Effect |
|------------|----------------------|--------|
| **Phrase** | For generic terms (hosting, free, how to, jobs) | Blocks any query containing the phrase in order; flexible enough to catch variations |
| **Exact** | For brand names (Wix, Squarespace, Bluehost) and competitor names | Surgical — only blocks that exact search. Prevents accidental blocking of "wix website help Dumfries" positive keyword |
| **Broad** | Not recommended for negatives in a small campaign | Too aggressive; can block valid queries unintentionally |

## Implementation Guidance

### Where to Apply

- **Account-level list:** Categories 5 (Job Seeking), 4 (Informational/Free/Cheap), and the generic parts of 1 (DIY — how to, tutorial, template) — these waste money in every campaign, not just web design.
- **Campaign-level:** Categories 2 (Hosting), 3 (Builder brands), 6 (Unrelated Tech), 7 (Competitors), 8 (Free Tools) — applied to the web design campaign specifically.
- **Ad group level:** Fine-tune if one ad group targets e-commerce clients — in that case, do NOT apply [shopify] or [bigcommerce] as negatives at that level.

### Ongoing Maintenance

- Review the Search Terms report **weekly** for the first 4 weeks, then fortnightly once stable.
- At £7/day budget, with the £400 match credit, you will accumulate enough query data to refine within 2–3 weeks.
- Any term that has generated 2+ clicks with zero conversions and shows informational/DIY intent → add to this list immediately.

### Wix Warning (Repeat)

"wix" appears as a POSITIVE keyword in your campaign (`wix website help Dumfries`). Do not add broad or phrase match negatives for "wix" — this will suppress your own warm-lead keyword. Use only exact match negatives for the pure DIY variants: `[wix]`, `[wix free]`, `[wix pricing]`, `[wix tutorial]`, `[wix templates]`, `[wix review]`.
