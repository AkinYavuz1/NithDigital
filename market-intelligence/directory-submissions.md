## How to Use This

Work through the list in priority order. Mark each row [x] when complete. For each listing, use the exact NAP below — copy-paste, don't retype.

**Exact NAP to use everywhere:**
```
Business name:  Nith Digital
Address:        Dumfries & Galloway, Scotland
Phone:          07[xxx] [xxx] [xxx]
Website:        https://nithdigital.uk
Email:          hello@nithdigital.uk
```

Never vary this. Not "Nith Digital Ltd." Not "nithdigital.uk" without the https. Same phone format every time.
