About Adkin Joinery

Adkin Joinery is run by Oliver Adkin, a skilled joiner based in Closeburn, Thornhill. Oliver works across Dumfries & Galloway, bringing the same care and attention to every project — whether it's a full house build or a set of shelves.

Every piece of work is built by hand. No shortcuts, no subcontractors, no templated solutions. Just honest joinery done properly.
